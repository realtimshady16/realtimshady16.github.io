README Contributor Tool ✨

This tool helps you contribute to a GitHub project's README directly from your browser. You'll need a Personal Access Token (PAT) with 'repo' scope.

Create an account on Github. Then click your profile icon and go to Settings > Developer Settings (at the bottom of 'Archives')> Personal access tokens > Tokens (classic) > Generate new token > Select the 'classic' option > Select the 'repo' checkbox > Copy the token and store it somewhere, as you only get to see it once!
